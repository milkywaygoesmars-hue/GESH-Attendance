🔧 Hardware & Setup Guide

✅ This system works offline. All data is stored on the device. Internet is only needed for Google Sheets sync.
📱
Tablet or Smartphone
Android or iOS tablet preferred for site use. A dedicated tablet mounted at the site entrance gate works best. Any modern browser works.
Required
📷
Rear Camera
Device must have a working rear camera for scanning helmet QR codes. Any smartphone or tablet from 2018+ works fine.
Required
🪖
QR Stickers on Helmets
Print QR codes from the Worker Registry tab. Laminate and stick them on each worker's safety helmet. Use weatherproof labels for outdoor sites.
Required
🖨
Label Printer
For printing QR stickers. A Brother QL or Zebra label printer produces durable outdoor-grade sticker labels. A4 printer works too.
Required
📡
Mobile Hotspot / WiFi
Only needed for Google Sheets sync. All attendance recording works offline. Use a 4G hotspot for remote site syncing when convenient.
Optional
🔋
Portable Power Bank
For outdoor/field use where power outlets are unavailable. A 20,000mAh power bank can run a tablet for a full work day.
Optional
💾
Offline Storage
All records stored in browser local storage. Supports hundreds of sessions. Export to PDF/Google Sheets regularly as backup.
Built-in
📊
Google Sheets Integration
Set up a Google Apps Script Web App to receive data. Paste your script URL in Settings below. All sessions sync automatically when online.
Optional

⚙ Google Sheets Webhook URL

Paste your Google Apps Script Web App URL below. Data will be sent as JSON when you click "Send to Google Sheet".

Google Apps Script (paste in script.google.com):

function doPost(e) {
  var data = JSON.parse(e.postData.contents);
  var ss = SpreadsheetApp.getActiveSpreadsheet();
  var sheet = ss.getSheetByName(data.type) || ss.insertSheet(data.type);
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(['Date','Time','Type','Topic','Conductor','#','Name','Designation','Package','Timestamp']);
  }
  data.attendees.forEach(function(a) {
    sheet.appendRow([data.date,data.time,data.type,data.topic,data.conductor,a.num,a.name,a.designation,a.package,a.scannedAt]);
  });
  return ContentService.createTextOutput('OK');
}

📋 Step-by-Step Workflow

  1. Register workers in the Worker Registry tab → Generate their QR codes
  2. Print & laminate the QR code stickers → Stick on each worker's safety helmet
  3. On briefing day, go to Take Attendance tab → Fill in session details → Start Session
  4. Use the QR Scanner tab to scan helmets as workers arrive, OR let workers scan themselves if a tablet is on a stand
  5. Take group photos as evidence → Upload in the Photo Evidence section
  6. When done, Save Session → Preview in Print tab → Print for physical signatures
  7. Once signed, scan/upload the signed copy for digital record (optional)
  8. Sync to Google Sheets for central record keeping when internet is available
